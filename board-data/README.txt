BOARD MEMBER DATA TEMPLATE
===========================

Fill out this template for each board member and save as board-member-NAME.txt (e.g., board-member-john-doe.txt)

REQUIRED FIELDS (must be filled):
BOARD MEMBER NAME: John Doe
BOARD MEMBER TITLE: President
BOARD MEMBER IMAGE: board-member-john-doe.jpg

OPTIONAL FIELDS (can be omitted if not applicable):
BOARD MEMBER BIO: Brief description of their role and background in the association.
BOARD MEMBER EMAIL: john.doe@example.com
BOARD MEMBER LINKEDIN: https://linkedin.com/in/johndoe
BOARD MEMBER YEAR JOINED: 2023
BOARD MEMBER PREVIOUS TEAM: 2022 Team Member

INSTRUCTIONS:
1. Copy this template for each board member
2. Fill in the actual information
3. Save as board-member-NAME.txt (replace NAME with their name, use hyphens for spaces)
4. Place board member images in the images/ folder
5. Run the update script to regenerate the association page

EXAMPLE WITH ALL FIELDS:
BOARD MEMBER NAME: Sarah Johnson
BOARD MEMBER TITLE: President
BOARD MEMBER IMAGE: board-member-sarah-johnson.jpg
BOARD MEMBER BIO: Sarah is a former iGEM team member who now leads the association. She has extensive experience in synthetic biology and project management.
BOARD MEMBER EMAIL: sarah.johnson@example.com
BOARD MEMBER LINKEDIN: https://linkedin.com/in/sarahjohnson
BOARD MEMBER YEAR JOINED: 2023
BOARD MEMBER PREVIOUS TEAM: 2022 Team Member

EXAMPLE WITH MINIMAL FIELDS (only required):
BOARD MEMBER NAME: Alex Chen
BOARD MEMBER TITLE: Treasurer
BOARD MEMBER IMAGE: board-member-treasurer.jpg
